# Enterprise Hotel Network Infrastructure Deployment

## Overview
Demonstrated hands-on experience designing and deploying a simulated enterprise hotel network infrastructure using Cisco Packet Tracer. This project involved designing a three-floor network topology, implementing VLAN-based network segmentation, configuring Layer 2 and Layer 3 network services, deploying dynamic routing, securing administrative access, implementing wireless networking, and validating end-to-end network connectivity across the enterprise environment.

This project demonstrates practical experience with enterprise network infrastructure deployment, VLAN implementation, inter-VLAN routing, dynamic routing protocols, network security, wireless networking, DHCP services, and enterprise network validation.

---

## Environment / Tech Stack
- Cisco Packet Tracer
- Cisco 2901 Integrated Services Routers
- Cisco Catalyst 2960 Switches
- Cisco Access Points
- PC-PT
- Printer-PT
- Laptop-PT
- Smartphone-PT

---

## Enterprise Network Infrastructure Deployment
- Designed a three-floor enterprise hotel network topology
- Deployed routers, switches, access points, workstations, laptops, smartphones, and network printers
- Established LAN and WAN connectivity using Ethernet and Serial DCE connections
- Segmented departmental traffic using VLANs
- Configured IEEE 802.1Q trunk links between switches and routers
- Implemented Router-on-a-Stick for inter-VLAN routing
- Configured DHCP services for automatic IP address assignment across all VLANs
- Implemented OSPF dynamic routing between floor routers
- Configured SSH for secure remote router administration
- Implemented switch port security using sticky MAC address learning with violation shutdown mode
- Deployed WPA2-PSK secured wireless networks across all floors
- Connected wireless clients to their assigned wireless networks
- Configured end-user devices to obtain IP addressing dynamically through DHCP
- Validated IP addressing, network connectivity, and secure remote administration

## Key Takeaways
Enterprise network infrastructure relies on proper network segmentation, scalable routing, secure administrative access, centralized IP address management, and comprehensive validation. This project demonstrates how VLANs, inter-VLAN routing, OSPF, DHCP, SSH, wireless networking, and switch security integrate to provide a secure, scalable, and manageable enterprise network.
