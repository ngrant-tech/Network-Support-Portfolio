# Network Support Portfolio

Enterprise-focused network infrastructure portfolio demonstrating hands-on experience designing, deploying, and validating Cisco-based enterprise networks. Projects showcase practical implementation of routing and switching, VLAN segmentation, Layer 3 switching, inter-VLAN routing, dynamic routing, DHCP services, network security, wireless networking, and enterprise infrastructure deployment.

Projects simulate real-world enterprise networking scenarios using Cisco Packet Tracer and demonstrate practical experience with enterprise LAN/WAN implementation, hierarchical campus architecture, multi-site connectivity, network segmentation, infrastructure validation, and end-to-end network communication.
